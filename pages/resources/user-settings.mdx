### Legacy Guild Folder Object

###### Legacy Guild Folder Structure

| Field     | Type             | Description                                                                              |
| --------- | ---------------- | ---------------------------------------------------------------------------------------- |
| color     | integer          | The color of the folder encoded as an integer representation of hexadecimal color code   |
| guild_ids | array[snowflake] | The IDs of guilds that are in this folder                                                |
| id        | ?integer         | The ID of the folder                                                                     |
| name      | ?string          | The name of the folders (defaults to `', '.join(guild.name for guild in folder.guilds)`) |
